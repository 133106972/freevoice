# 微信小程序示例

这是一个简化版的小程序前端代码示例，用于配合 FastAPI 后端实现文本转语音功能。

## ⚠️ 重要说明

### 1. 服务器域名配置

在微信小程序后台（微信公众平台）配置服务器域名白名单：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「开发」→「开发管理」→「开发设置」
3. 在「服务器域名」中添加：
   - request 合法域名: `https://你的域名`
   - downloadFile 合法域名: `https://你的域名`

### 2. 后端服务要求

- 必须使用 **HTTPS** 协议
- 域名必须备案（国内服务器）
- 需要在小程序后台配置域名白名单

### 3. 本地开发调试

在微信开发者工具中，可以开启「不校验合法域名」进行本地调试：

1. 点击右上角「详情」
2. 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」

## 🚀 使用步骤

1. 将 `mini_program` 文件夹导入微信开发者工具
2. 修改 `index.js` 中的 API 地址为你的后端服务地址
3. 配置服务器域名白名单（上线前必须）
4. 编译预览

## 🔗 后端接口

小程序调用的是同一个 FastAPI 后端接口：

```
POST /api/tts
Content-Type: application/json

{
  "text": "要转换的文本",
  "voice": "zh_female_qingxin",
  "speed": 1.0,
  "volume": 1.0
}
```

## 💡 注意事项

1. 小程序有 2MB 的代码包大小限制
2. 音频文件建议先下载到本地再播放
3. 需要考虑用户未授权音频播放权限的情况
